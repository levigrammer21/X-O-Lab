Patch Notes

Current Version: v1.1.2

Current Build
v1.1.2 — Ball Timing Control
  • Snap now instantly places the ball on the first selected non-center player.
  • Handoff, pitch, and instant transfers now behave as possession changes instead of floating paths.
  • Advanced ball controls now include exact transfer timing chips for bang-bang handoffs, delayed reverses, and passes.
  • Pass remains the only transfer type where the ball travels through the air.
v1.1.1 — Ball Attachment Fix
  • Football now stays attached to a player instead of floating between players.
  • Ball stays still during pre-snap motion and begins after the snap phase starts.
  • Non-pass transfers now work like instant possession changes.
  • Advanced ball type includes Instant transfer; Pass remains an airborne throw.
v1.1.0 — HD Foundation
  • HD visual overhaul for a sharper premium app feel.
  • Improved field rendering, player icons, routes, and play previews.
  • Play Test fit patch so diagrams show fully with compact notes.
  • Ball progression foundation with Basic and Advanced settings.
  • Version badge and persistent changelog system added.
v1.0.0 — Initial Public Build
  • Playbooks, formations, and play editor.
  • Player routes, motion, blocks, zones, and animation.
  • Play Test and leaderboard foundation.
  • View-code access for players and parents.
  • Email, Google, Apple, guest, and legacy coach access paths.
